### Result
```Launch Telegram and Search for @zappru_bot
You can control Bot by sending these commands:
    /help - To get command 
    /start - Codelocker Bot
    
Server Details:

    /servertime - to get Server time and date         
    /listfile - list out files on server   
    /detail - server details   
    /hello - Test command

    /get btc
    /get eth
```
